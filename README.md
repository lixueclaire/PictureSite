Picture Site
==============================

This is a website to display photos. In this project, you will use:
* web.py
* bootstrap3
* carousel, masonry
* postgresql

Install
------------

1. Install Web.py
2. Install postgreSQL
3. Install psycopg2


Run
------------

1. Run
```
nohup python index.py 80 &
```
2. Visit by 0.0.0.0:80
